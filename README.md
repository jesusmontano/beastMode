# beastMode

BeastMode is a customized workout generator for the beginner to experienced athlete. The application features a workout form to create a workout depending on the muscle group, access to equipment, and level of fatigue.

[Live Demo](http://beastmodepro.herokuapp.com/#/)

### Technologies
* React
* Redux
* MongoDB
* Express
* Node
* HTML5
* CSS3
* Heroku
* AWS

## Login Features

## Workout Page
<img width="1440" alt="Workout-Page" src="https://user-images.githubusercontent.com/7242067/63716205-467f2580-c7fa-11e9-821f-259a4424b7f1.png">
* The workout page includes exercises depending on the workout that the application's algorithm has designed. The page includes three exercises with detailed instructions including number of sets and repetition. Users are required to rate workouts after their exercise, describing their experience.

## Profile Page
<img width="1440" alt="Profile" src="https://user-images.githubusercontent.com/7242067/63716202-45e68f00-c7fa-11e9-8ba6-272cdc29e6ea.png">
* The profile page includes all of the user's previous workouts, detailing the exercises and the ratings for each workout. Users can view previous workouts for reference and to see their progression and experience.
