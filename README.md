# beastMode

BeastMode is a customized workout generator for the beginner to experienced athlete.

[Live Demo](http://beastmodepro.herokuapp.com/#/)

### Technologies
* React
* MongoDB
* Express
* Node
* HTML5
* CSS3
* Heroku
* AWS
